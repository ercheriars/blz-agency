# BLZ Agency | Future-Forward Digital Design

![BLZ Agency Logo](logo.png)

BLZ Agency is a premium, growth-focused creative agency that blends futuristic aesthetics with strategic digital solutions. We help brands, businesses, and founders dominate the digital era through cinematic storytelling and high-impact design.

## ✨ Core Specializations

- **Personal Branding**: Building authority and trust for CEOs and professionals.
- **Social Media Management**: Strategic growth and consistency for modern brands.
- **Real Estate Content Production**: Cinematic property showcases and luxury branding.
- **Design Branding**: Holographic-inspired visual identities and premium design systems.

## 🚀 Key Features

- **Holographic Aesthetic**: A custom design system built around dark modes, vibrant gradients, and fluid animations.
- **Glassmorphism**: Elegant, translucent UI components that provide a modern, depth-focused user experience.
- **Interactive Project Showcase**: A dynamic portfolio grid with hover-reveal overlays and smooth scaling transitions.
- **Cinematic Integration**: Integrated high-definition company profile videos and hero "cosmic arcs."
- **Optimized Performance**: Fast-loading, responsive design tailored for high-end digital discovery.

## 🛠 Technology Stack

- **Core**: Semantic HTML5 & Modern CSS3
- **Animations**: CSS Keyframes & Vanilla JavaScript
- **Typography**: TAN PEARL (Custom Serif) & Raleway (Clean Sans-Serif)
- **Deployment**: Optimized for Vercel with clean URL routing

## 📦 Getting Started

### Prerequisites
- Any modern web browser
- A local server environment (optional, but recommended for video playback)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/blz-agency.git
   ```
2. Open `index.html` in your browser or run a local server:
   ```bash
   # If using Python
   python3 -m http.server 8000
   ```

## 🌐 Deployment

This project is configured for seamless deployment on **Vercel**. It includes a `vercel.json` for clean URL handling:

```json
{
  "cleanUrls": true
}
```
